# Ansible Role mafalb.scl.rh

Make the Software Collections (SCL) available
This is only supported on RHEL7 and Clones

## Basic Usage

```yaml
- name: install mafalb.scl.rh
  hosts: localhost
  roles:
  - role: mafalb.scl.rh
```
